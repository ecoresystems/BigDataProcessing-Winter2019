How to use cloud environment クラウド環境の使い方

Use our Jupyter hub    Jupyterハブを使用する

You can either enter url http://133.5.19.187:8080 directly, or click on "Go to JupyterHub" on Course Material page as shown below.

Url http://133.5.19.187:8080 を直接入力するか、以下に示すように[コース資料]ページで[JupyterHub]をクリックします。

The username is your student ID with all the letters in lower case

ユーザー名は、すべての文字が小文字の学籍番号です。

Upon your first log-in attempt, whatever password you use will be stored as your password for subsequent login attempts.

最初のログイン試行時に、使用するパスワードは、以降のログインのパスワードとして保存されます。

Use sample notebook

サンプルノートブックを使用する

To use the sample notebook we've created for you, go to the course material page and click on the corresponding week's material.

作成したサンプルノートブックを使用するには、コース資料ページに移動し、対応する週の資料をクリックします。

This will redirect you to our JupyterHub with materials automatically copied to your cloud workspace.

これにより、JupyterHubにリダイレクトされ、サンプルノートブックがクラウドワークスペースに自動的にコピーされます。

IMPORTANT! This process may overwrite the file with the same name, be sure to save your work with a different name.

重要！ この操作は、同じ名前のファイルを上書きする可能性がありますので、必ず別の名前で作業を保存してください。
